# AWS EC2 Static Website Hosting

A simple cloud project demonstrating how to deploy and host a static website on an AWS EC2 Linux instance using Apache HTTP Server.

## What I Did
- Launched an AWS EC2 instance
- Installed and configured Apache HTTP Server
- Deployed website files using SCP with PEM key authentication
- Hosted the site using EC2 public IP
- Managed instance start/stop to optimize cost

## Tech Stack
AWS EC2 • Linux • Apache • HTML • CSS • JavaScript • SCP

## Access
The website is accessible via the EC2 public IP while the instance is running.
